This source code requires a few things to be installed to edit or launch due to it not being packaged.

1. Visual Studio Code Community: https://visualstudio.microsoft.com/downloads/
2. Love2d Game Engine: https://love2d.org/
3. Lua: https://www.lua.org/

It is highly recomended to use the following extension in visual studio code for editing Lua and Love2d code

1. Love2D Support: https://marketplace.visualstudio.com/items?itemName=pixelbyte-studios.pixelbyte-love2d
2. vscode-lua: https://marketplace.visualstudio.com/items?itemName=trixnz.vscode-lua

these can be installed inside Visual Studio Code using the extensions tab on the left had side.
